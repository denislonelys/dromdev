Установка IIStudio

AI Dev Tool с Claude Opus 4.6 и Sonnet 4.6

1

Установи Python и CLI

Нужен Python 3.10+ (Ubuntu/Debian/Mac/Windows WSL)
git clone https://github.com/denislonelys/dromdev IIStudio cd IIStudio && bash install.sh
2

Зарегистрируйся и получи токен

Зайди в личный кабинет → скопируй API токен (sk-iis-...)

Войти / Зарегистрироваться
После входа токен будет в кабинете
3

Войди в CLI и начни работу

# Войди с токеном из кабинета: iis auth login --token sk-iis-... # Спроси AI: iis ask "напиши hello world на Python" # Интерактивный чат: iis chat

Все команды CLI

Запросы к AI
iis ask "вопрос"
iis chat
iis ask --model claude-opus-4-6 "..."
iis ask --stream "..."
Инструменты
iis plan "задача"
iis fix "описание бага"
iis review .
iis yolo "создай проект"
Задачи
iis tasks
iis task add "описание #тег"
iis task done ID
Аккаунт
iis auth login
iis auth register
iis auth status
iis version
