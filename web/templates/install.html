Установка IIStudio

AI Dev Tool с Claude Opus 4.6 и Sonnet 4.6

1

Установи Python и CLI

Нужен Python 3.10+ (Ubuntu/Debian/Mac/Windows WSL)
# Установка (Ubuntu/Debian/Mac): git clone https://github.com/denislonelys/dromdev IIStudio cd IIStudio && bash install.sh
2

Зарегистрируйся и получи токен

Зайди в личный кабинет → скопируй API токен (sk-iis-...)

Войти / Зарегистрироваться
После входа токен будет в кабинете
3

Войди в CLI и начни работу

# Войди (интерактивно — введи email и токен): iis dromdev auth login # Запусти агента: iis dromdev run # Или одиночный запрос: iis dromdev ask "напиши hello world на Python"

Все команды CLI

Основные
iis dromdev run
iis dromdev ask "вопрос"
iis dromdev ask -m claude-sonnet-4-6 "..."
iis dromdev ask --stream "..."
Аккаунт
iis dromdev auth login
iis dromdev auth logout
iis dromdev auth status
iis version
Инструменты
iis dromdev plan "задача"
iis dromdev fix "баг"
iis dromdev review .
iis dromdev yolo "создай проект"
Задачи
iis dromdev tasks
iis task add "описание #тег"
iis task done ID
iis dromdev models
